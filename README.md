# Numbers Project

Проект содержит набор функций для работы с числами на Python.

## Функции

- `sum_digits(number)` — сумма цифр числа.
- `count_digits(number)` — количество цифр в числе.
- `max_digit(number)` — максимальная цифра числа.
- `min_digit(number)` — минимальная цифра числа.
- `is_even(number)` — проверка на чётность.
- `multiply_digits(number)` — произведение цифр числа.
- `reverse_number(number)` — число, перевёрнутое задом наперёд.
- `is_palindrome(number)` — проверка, является ли число палиндромом.
- `count_even_digits(number)` — количество чётных цифр.
- `count_odd_digits(number)` — количество нечётных цифр.
- `first_digit(number)` — вернуть первую цифру числа number.
- `last_digit(number)` — вернуть последнюю цифру числа number.
- `sum_even_digits(number)` — посчитать сумму чётных цифр в числе number.
- `sum_odd_digits(number)` — посчитать сумму нечётных цифр в числе number.
- `square_number(number)` — вернуть квадрат числа number.
- `cube_number(number)` — вернуть куб числа number.
- `average_digit(number)` — вернуть среднее арифметическое цифр числа number.
- `is_positive(number)` — проверяет, является ли число number положительным.
- `is_negative(number)` — проверяет, является ли число number отрицательным.

## Запуск

```bash
python3 numbers_lib.py
```
